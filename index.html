YOUTUBE DATA API

Aramayı etkinleştir

YouTube’da şarkı aramak için API anahtarını bir kez ekle. Anahtar alan adına göre kısıtlandıysa yalnızca bu sitede çalışır.

  1. Google Cloud Console’da bir proje oluştur.
  2. YouTube Data API v3 hizmetini etkinleştir.
  3. Credentials bölümünden bir API key oluştur.
  4. GitHub Pages’e koyunca anahtarı sitenin alan adına göre HTTP referrer ile kısıtla.
GROQ MÜZİK AJANI

Asistan bağlantısı

Groq anahtarını bu cihazda saklayabilir veya daha sonra güvenli bir sunucu bağlantısı kullanabilirsin.

Henüz bir Groq anahtarı kaydedilmedi.
GitHub Pages herkese açık olduğu için Groq anahtarını HTML koduna yazmak güvenli değildir. Bu alan anahtarı yalnızca bu tarayıcının yerel depolamasında tutar. Kalıcı kullanımda küçük bir sunucu veya Cloudflare Worker üzerinden proxy kullanmak daha doğrudur.
Ajan yalnızca tanımlı müzik araçlarını çalıştırır; rastgele JavaScript veya HTML kodu çalıştıramaz.
Albüm toplamak için YouTube Data API anahtarının da etkin olması gerekir.

Dosyayı web sunucusundan aç

YouTube oynatıcısı file:// adresinde güvenli JavaScript iletişimi kuramaz. Bu klasörde terminal açıp aşağıdaki komutu çalıştır:

python3 -m http.server 8765 --bind 127.0.0.1

Ardından tarayıcıdan http://127.0.0.1:8765/ adresini aç. Paketle gelen baslat.sh dosyası bunu otomatik yapar.
