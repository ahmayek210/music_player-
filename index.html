YOUTUBE DATA API

Aramayı etkinleştir

Hazır dört şarkı anahtarsız oynar. Uygulama içinde istediğin şarkıyı aramak için kendi ücretsiz API anahtarını ekle.

  1. Google Cloud Console’da bir proje oluştur.
  2. YouTube Data API v3 hizmetini etkinleştir.
  3. Credentials bölümünden bir API key oluştur.
  4. GitHub Pages’e koyunca anahtarı sitenin alan adına göre HTTP referrer ile kısıtla.

Dosyayı web sunucusundan aç

YouTube oynatıcısı file:// adresinde güvenli JavaScript iletişimi kuramaz. Bu klasörde terminal açıp aşağıdaki komutu çalıştır:

python3 -m http.server 8765 --bind 127.0.0.1

Ardından tarayıcıdan http://127.0.0.1:8765/ adresini aç. Paketle gelen baslat.sh dosyası bunu otomatik yapar.
